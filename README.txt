FUNCIONES UNA POR UNA: SUPERVISORES: Acà el supervisor debera "cargar" las imagenes que desee, completando todos los campos. Se cargan imagenes individualmente. Boton eliminar, el supervisor puede eliminar y cargar imagenes las veces que se requiera. Las imagenes se guardan de manera remota. Boton bees, se deben cambiar parametros, pero este boton te abre la aplicacion bees de forma local.

PROMOTORES: Los promotores pueden visualizar todas las promociones, listas y tambien de agregado, un sector llamado incentivos, donde se podran visualizar y llevar un registro de todos los incentivos activos.

CLIENTES: Los clienten pueden visualizar todas las promociones, secciones de bees, soporte, y listas segun su clasificacion. Tambien tiene botones para entrar a bees y un whatsapp del promotor.

PAGINA WEB INGRESO: Se realiza la clasificaion, otorgando una palabra clave, se podra acceder al archivo correspondiente.
